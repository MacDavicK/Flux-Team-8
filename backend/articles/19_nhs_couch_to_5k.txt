Category: cardio | Authority: government

Couch to 5K is a running plan for absolute beginners. It was developed by a new runner, Josh Clark, who wanted to help his 50-something mum get off the couch and start running. The NHS Couch to 5K programme has helped millions of people around the world start running, often for the first time in their lives.

The plan is designed to get just about anyone from the sofa to running 5 kilometres (or 30 minutes) in just 9 weeks. The secret is starting gently and building up gradually, so the challenge never feels too great.

How the Plan Works

The programme is structured into 3 runs per week with a rest day between each. Each run begins with a 5-minute warm-up walk and ends with a 5-minute cool-down walk.

During the runs, you alternate between walking and running. Each week, the running intervals get slightly longer and the walking intervals get shorter, until you're running continuously for 30 minutes by Week 9.

Tips for Success

Don't skip rest days: Recovery is essential. Your body adapts to the new demands of running during rest, not during training. Running 3 days a week is enough.

Go slowly: Many beginners run too fast, which makes running feel hard and uncomfortable. You should be able to hold a conversation while running — if you can't, slow down.

It's OK to repeat a week: If a week feels too hard, it's perfectly fine to repeat it before moving on. Progress is not linear, and building confidence is as important as building fitness.

Get good shoes: Proper running shoes make a significant difference in comfort and injury prevention. Visit a specialist running shop for a gait analysis if possible.

Stay hydrated: Drink water before, during (for longer runs), and after your run.

9-Week Programme Overview

Week 1: Walk 5 min → alternate 60 sec run / 90 sec walk for 20 min → Walk 5 min (8 run intervals)
Week 2: Walk 5 min → alternate 90 sec run / 2 min walk for 20 min → Walk 5 min (6 run intervals)
Week 3: Walk 5 min → 2 × (90 sec run / 90 sec walk / 3 min run / 3 min walk) → Walk 5 min
Week 4: Walk 5 min → 3 min run / 90 sec walk / 5 min run / 2.5 min walk / 3 min run / 90 sec walk / 5 min run → Walk 5 min
Week 5:
- Run 1: Walk 5 min → 5 min run / 3 min walk / 5 min run / 3 min walk / 5 min run → Walk 5 min
- Run 2: Walk 5 min → 8 min run / 5 min walk / 8 min run → Walk 5 min
- Run 3: Walk 5 min → 20 min continuous run → Walk 5 min
Week 6:
- Run 1: Walk 5 min → 5 min run / 3 min walk / 8 min run / 3 min walk / 5 min run → Walk 5 min
- Run 2: Walk 5 min → 10 min run / 3 min walk / 10 min run → Walk 5 min
- Run 3: Walk 5 min → 22 min continuous run → Walk 5 min
Week 7: Walk 5 min → 25 min continuous run → Walk 5 min (all 3 sessions)
Week 8: Walk 5 min → 28 min continuous run → Walk 5 min (all 3 sessions)
Week 9: Walk 5 min → 30 min continuous run → Walk 5 min (all 3 sessions)

Running Form for Beginners

Good form makes running easier and reduces injury risk:

- Head: look ahead, not down at your feet; gaze at the horizon or about 10–15 metres ahead
- Shoulders: relaxed and low, not hunched up around your ears
- Arms: bent at approximately 90 degrees, swinging forward and back (not across your body)
- Hands: loosely curled, as if holding a crisp without crushing it
- Core: gently engaged, body tall and upright
- Hips: level, not rocking from side to side
- Feet: landing with your foot underneath your hips, not far out in front of you
- Cadence: 160–180 steps per minute is generally most efficient (try counting your steps for 30 seconds and multiplying by 2)

Common Beginner Injuries and Prevention

Shin splints: Pain along the shinbone, common in new runners. Caused by doing too much too soon. Prevention: follow the programme as written, don't increase distance too rapidly, and run on softer surfaces when possible.

Knee pain (runner's knee): Discomfort around the kneecap. Often related to weak hip and glute muscles. Prevention: include glute-strengthening exercises alongside running, and ensure good running form.

Achilles tendinopathy: Stiffness or pain in the Achilles tendon, usually in the first few steps of the morning. Prevention: avoid rapid increases in running distance, wear appropriate footwear.

Blisters: Caused by friction between sock and skin. Prevention: wear moisture-wicking running socks and ensure shoes fit properly.

If pain is sharp, sudden, or doesn't resolve with rest, stop running and seek advice.

Why Running is Worthwhile

Running is one of the most time-efficient aerobic exercises available. Regular running:
- Significantly improves cardiovascular fitness
- Burns approximately 400–600 calories per 5 km (depending on body weight and pace)
- Improves insulin sensitivity
- Reduces resting heart rate and blood pressure
- Releases endorphins and improves mood
- Can be done anywhere with no equipment beyond suitable footwear
- Builds community — parkrun events make running social and accessible

Completing your first 5K is a significant milestone, but many graduates find that running becomes a lifelong habit that supports both physical and mental health.
